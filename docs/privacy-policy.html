Privacy Policy

OpenClaw Agent Workspace Monitor — Last updated: March 10, 2026

Overview

OpenClaw Agent Workspace Monitor is a Chrome extension that visualizes AI agent activity as a pixel art office simulation. We are committed to protecting your privacy.

Data Collection

This extension does not collect, transmit, or store any personal data. Specifically:

  • No personal information is collected.
  • No browsing history or web activity is tracked.
  • No analytics, telemetry, or crash reports are sent to any server.
  • No cookies are set by the extension.
  • No data is sold or shared with third parties.

Local Storage

The extension uses Chrome's storage API to save your preferences locally on your device (e.g., server URL, notification settings). This data never leaves your browser.

Network Communication

The extension connects to a locally running OpenClaw agent server (typically on localhost) via WebSocket to receive real-time agent status updates. No data is sent to any external or remote server.

Permissions

  • sidePanel — Renders the office simulation UI.
  • storage — Saves user preferences locally.
  • notifications — Alerts when agent status changes.
  • alarms — Schedules periodic reconnection to the local server.

Changes
