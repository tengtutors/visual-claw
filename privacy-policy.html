Visual Claw Privacy Policy

Effective date: March 11, 2026

Visual Claw is a Chrome extension that visualizes OpenClaw agent activity in a live pixel-art workspace monitor. This Privacy Policy explains what data the extension uses and how that data is handled.

Information Visual Claw Uses

Visual Claw may use the following information only for its core functionality:

  • Local extension storage for preferences and monitor state
  • Agent status and event data supplied by a local or configured OpenClaw service
  • Side panel UI state needed to display the workspace monitor
  • Alarm and notification state used for extension behavior

How Information Is Used

Visual Claw uses data only to provide the extension’s core features, including:

  • Rendering the office monitor and agent states
  • Displaying event history and agent detail views
  • Saving local extension preferences
  • Supporting polling, refresh, and notification behavior

What Visual Claw Does Not Do

  • Does not sell personal data
  • Does not use data for advertising or marketing
  • Does not transfer data to data brokers
  • Does not collect browsing history for unrelated purposes
  • Does not use data for creditworthiness or lending decisions

Data Sharing

Visual Claw does not sell or share personal data for advertising. Any agent or event data used by the extension is handled only to support the product’s monitoring features.

Data Retention

Data stored by the extension remains locally available until cleared by the user, removed by extension reset behavior, or deleted when the extension is uninstalled.

Remote Communication

Visual Claw may connect to a local or configured OpenClaw service to retrieve live agent data required for the monitor experience.

User Controls

  • You can remove the extension at any time
  • You can clear extension data through Chrome’s extension controls
